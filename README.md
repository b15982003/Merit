#  🤖 Merit（功德）

《Merit》 是一款融合了傳統與惡趣味的點擊系娛樂 APP
每次點擊螢幕中央的木魚，不只聽見療癒音效，還會隨機出現一句道德金句——
有的嚴肅、有的搞笑，有的讓你懷疑人生。
是修行？還是發洩？誰知道呢，開心就好！
